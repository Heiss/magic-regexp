Expand description

Represents a regex type. This enum is used to create the smallest regex statement. For example, Type::Digit will create the regex \d.

Examples

use magic_regexp::{OneOrMore, Type::Digit};

let input = OneOrMore(Digit);
assert_eq!(input.to_string(), r"(\d+)"); // Note that the regex is wrapped in parentheses.
